All About Custom Keyboards

Step 1: Finding the Right Size Keyboard

Here is a quick overview of all of the different keyboard sizes:
  • Full-sized - You usually can’t find this size for custom keyboards, it’s only used on pre-built keyboard
  • 75% - The navigational cluster is placed vertically to save space. Fun and comfortable to use
  • 65% - This layout removes the function row and keeps some of the navigational cluster
  • 60% - The most common custom keyboard layout. There is no function row, arrow keys, or navigational cluster
  • 40% - The smallest size that you can still type on. It’s basically a 60% keyboard except you remove the numeric keys
